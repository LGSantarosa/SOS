# SOS – Frontend

Interface em HTML e JavaScript (sem framework).

## Estrutura

- `index.html` – página principal
- `js/app.js` – lógica da aplicação

## Como rodar

1. **Direto no navegador:** abra `index.html`.
2. **Com servidor local (recomendado):**
   ```bash
   cd front
   python -m http.server 8080
   ```
   Acesse: http://localhost:8080
